# 🎓 Student Performance Prediction System

## 📖 Overview

This project is a Machine Learning web application that predicts a student's final grade (G3) based on academic, personal, and family-related information using a Linear Regression model.

---

## 🚀 Features

- 🎯 Predict Student Final Grade
- 📊 Machine Learning Model (Linear Regression)
- 🌐 Interactive Streamlit Web Application
- 📈 Real-time Prediction
- 💻 User-Friendly Interface

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib
- Matplotlib

---

## 📊 Machine Learning Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Preprocessing
5. Feature Engineering
6. Model Training
7. Model Evaluation
8. Model Deployment
